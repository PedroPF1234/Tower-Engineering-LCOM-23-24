typedef struct Player Player

Struct that represents the player.

Struct that contains the player game object, the sprites, the current direction, the position, the speed, the origin offset, the hit points and the weapon status.

Parameters
playerThe player game object.
spritesThe array of AnimatedSprites that represent the player walking in different directions.
current_directionThe current direction of the player.

origin_offset_yThe offset of the origin of the player sprite in the y direction.
