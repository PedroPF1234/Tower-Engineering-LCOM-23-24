Detailed Description

Struct that represents an animated sprite.

Struct that contains an array of sprites, the current sprite, the animation cooldown and the cooldown counter.

Parameters

current_spriteThe index of the current sprite.
animation_cooldownThe ammount of time between each sprite change (in milliseconds, ms).
cooldown_counterThe counter of the cooldown.
