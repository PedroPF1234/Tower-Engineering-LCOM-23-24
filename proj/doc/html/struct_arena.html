Detailed Description

Struct that represents an Arena.

Struct that contains the background sprite, the decorations, the towers, the shop and the player base.

Parameters
backgroundThe background sprite of the arena.
decorationsAn array of AnimatedGameObject pointers that represent the animated decorations on the arena.

shopThe shop object that is present in the arena.
baseThe player base object that is present in the arena.
num_targetsThe number of target coordinates that the enemies will follow.
num_decorationsThe number of decorations in the arena (Although variable, it is always 3).
target_coordinatesThe array of the coordinates that the enemies will follow.
